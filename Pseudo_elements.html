::first-line Pseudo-element

::first-line pseudo-element is used to add a special style to the first line of a text. The following properties apply to the ::first-line pseudo-element: font properties color properties background properties word-spacing letter-spacing text-decoration vertical-align text-transform line-height clear

::first-letter Pseudo-element

The ::first-letter pseudo-element is used to add a special style to the first letter of a text.
